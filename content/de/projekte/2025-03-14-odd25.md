## Was sind die HackDays? 

Der **Open Data Day** ist ein jährlich stattfindendes, internationales Event, das weltweit die Bedeutung offener Daten in den Fokus rückt. Überall organisieren lokale Gruppen eigene Veranstaltungen – so auch in Karlsruhe!
Hier finden die HackDays: Open Data for Urban Future statt. Gemeinsam wollen wir mit offenen Daten innovative Lösungen für urbane Herausforderungen entwickeln.

Der Open Data Hack Day ist nicht nur für Hacker oder IT-Profis! Wir orientieren uns an der Wikipedia-Definition von „Hacking“ – es geht darum, mit kreativen Methoden Lösungen für Herausforderungen zu finden.
Jede:r kann mitmachen – egal ob Anfänger:in oder Expert:in, unabhängig von Alter und technischer Erfahrung!

## ![alias="chart" style="width: 32px; display: inline-block !important;  background: none;  shadow: none; position: relative; bottom: -.5rem;  box-shadow: none !important; border: none !important; margin: 0 !important;"](/icons/chartUp.svg) Die Daten

Wir haben verschiedene Datenquellen angezapft:

 * Open Data Portale:
    * KA/Regional/BW
      *  [Transparenzportal Karlsruhe](https://transparenz.karlsruhe.de/dataset/)
      *  [Statistikportal Karlsruhe](https://web6.karlsruhe.de/Stadtentwicklung/statistik/atlas/)
      *  [Re@di Opendata Portal](https://opendata.readi.de/), Baden-Baden, Bretten, Bruchsal, Ettlingen, Rastatt, Rheinstetten, Stutensee
      *  [Open Data BW](https://www.daten-bw.de/)
      *  [Regionalstatistik](https://www.regionalstatistik.de/genesis/online/)

    *  Überregional
        * [GovData](https://www.govdata.de/) Hier gibt es auch ["High Value Datasets"](https://www.govdata.de/suche?tags=hvd)
        * [Destatis](https://www.destatis.de/DE/Home/_inhalt.html)
        *  Auswahl aus der Länderstistik: [Unfallatlas](https://unfallatlas.statistikportal.de)

  Neben diesen Portalen, die speziell auf die Bereitstellung von Daten ausgerichtet sind, gibt es weitere interessante Quellen, neben Verwaltung auch aus (Bürger-) Wissenschaft und Forschung. Hier liegen die Information häufig in besonderen Formaten vor und müssen ggf umgewandelt werden. 
  * [Ratsinformationssystem Karlsruhe](https://sitzungskalender.karlsruhe.de/db/ratsinformation/start)
  * 3D Daten Karlsruhe - zugänglich über das Transparenzportal, ggf Konvertierung notwendig  
  ![alias="Zoo3D" style="width: 128px; display: inline-block !important;  background: none;  shadow: none; position: relative; bottom: -.5rem;  box-shadow: none !important; border: none !important; margin: 0 !important;"](/projects/karlsruhe/odd2025/zoo3d.png)
  * Der *IDEAL Household Energy Dataset* des FZI enthält Daten zum Energieverbrauch aus 255 Haushalten im Vereinigten Königreich. 
  * [Next-Bike Stationsdaten](https://api.nextbike.net/maps/nextbike-live.json?countries=de) Die Online-Daten stehen nur für eine kurze Zeitspanne zur Verfügung. Der vorhandene Datensatz stellt eine Sammlung der Daten seit 2022 zur Verfügung.
  * [Autonomes Fahren](https://github.com/fzi-forschungszentrum-informatik/test-area-autonomous-driving-dataset) Mobilitätsdaten: Trajektoriendaten auf eine Forschungskreuzung – der TAF-BW Datensatz.  
  ![alias="TAF" style="width: 128px; display: inline-block !important;  background: none;  shadow: none; position: relative; bottom: -.5rem;  box-shadow: none !important; border: none !important; margin: 0 !important;"](/projects/karlsruhe/odd2025/taf.gif)  

  * [Stadtradeln 2023/2024](https://www.stadtradeln.de/home) - Daten zu gefahrenen Wegen in Karlsruhe  
  ![alias="Radeln" style="width: 128px; display: inline-block !important;  background: none;  shadow: none; position: relative; bottom: -.5rem;  box-shadow: none !important; border: none !important; margin: 0 !important;"](/projects/karlsruhe/odd2025/radeln.png)
  * ZKM Besucherbewegungen: Spezielle *Mobilitätsdaten* aus der Ausstellung *Renaissance 3.0* am ZKM. 
  Hieraus können sehr interessante Visualisierungen erzeugt werden.  
  ![alias="ZKM" style="width: 128px; display: inline-block !important;  background: none;  shadow: none; position: relative; bottom: -.5rem;  box-shadow: none !important; border: none !important; margin: 0 !important;"](/projects/karlsruhe/odd2025/zkm-tracks-small.jpg)
  * Das Klimaschutzkonzept Karlsruhe liefert einige Daten zu den geplanten Maßnahmen. Es gibt einen [Prototyp](https://dashboard.daten.city/) für 
  ein *Klimadashboard* um diese und weitere Daten anzuzeigen. 

  **Alle Datenquellen wurden von uns wenigstens auszugsweise abgerufen. Die Daten (und ggf Metadaten) liegen auf einem shared-Drive und können auf Eure Laptops kopiert werden.**
